## 2. КОПІ (заповни)

```
Headline (до 27 символів): 
Primary text (до 125 символів): 
CTA кнопка: [Get Free Quote / Check Availability / See Options]
Trust line (необов'язково): [напр. "4.9★ Google · BBB Accredited"]
```

**Правила копі (з copy-principles.md):**
- Знижка = потрібна причина + сума в доларах
- Urgency = конкретна дата або подія, не просто "limited time"
- Цифри завжди замість слів

## 4. ТЕХНІЧНІ ВИМОГИ ДЛЯ КЛОДА (фіксовані, не змінювати)

### Розміри та safe zones
| Формат | Canvas | Safe zone (текст/кнопки) |
|--------|--------|--------------------------|
| Feed 1:1 | 1080×1080px | внутрішній відступ мін. 80px з усіх сторін |
| Story 9:16 | 1080×1920px | верх/низ вільні 250px (UI overlay) |
| Landscape 1.91:1 | 1200×628px | відступ мін. 60px |

### Текстовий overlay
- Текст не займає більше 20% площі банера
- Без дрібного шрифту — мінімум 28px для body, 48px+ для headline
- Колір тексту завжди контрастний до фону (ratio ≥ 4.5:1)

### HTML/CSS вимоги
```
- Один HTML-файл, всі стилі inline або <style> блок
- Розмір canvas через: width/height в px + overflow: hidden
- Шрифти: Google Fonts (підключити через @import)
- Без зовнішніх зображень — тільки CSS shapes, градієнти, або base64
- Файл повинен коректно відкриватись у браузері без сервера
```

## 5. СТРУКТУРА БАНЕРА (стандартна)

Клод має дотримуватись цього layout-порядку:

```
[ LOGO / Бренд-лінія ]        ← верхній лівий, невеликий
[ HEADLINE ]                   ← центр, найбільший текст
[ SUBLINE або benefit ]        ← під headline
[ TRUST SIGNAL ]               ← рядок з зірками / BBB / кількість відгуків
[ CTA КНОПКА ]                 ← нижня частина, контрастний колір
[ OFFER TAG (якщо є) ]         ← банер-стрічка або бейдж
```
